# 角色权限矩阵（当前系统）

## 角色定义

- `STUDENT`（1，学生）
- `TEACHER`（2，心理咨询师）
- `ADMIN`（3，心理中心管理员，统一治理角色）

## 业务边界

- 学生：仅可访问本人相关数据，不可查看其他学生隐私数据。
- 心理咨询师：仅可处理分配给自己的预约与咨询，不可查看其他咨询师私有数据。
- 心理中心管理员：负责心理业务治理与系统治理（账号/角色/配置/日志），不参与具体咨询会话。

## 权限矩阵

| 功能域 | 学生 | 咨询师 | 心理中心管理员 |
|---|---|---|---|
| 登录 | 是 | 是 | 是 |
| 预约心理咨询 | 是 | 否 | 查看全部并监管 |
| 在线咨询 | 是（仅本人） | 是（仅分配给自己） | 否 |
| 匿名倾诉发帖/回复 | 是 | 可回复（按业务策略） | 举报审核与治理 |

| 心理科普查看 | 是 | 是 | 是 |
| 心理科普发布 | 否 | 是 | 否 |
| 心理科普审核 | 否 | 否 | 是 |
| 举报审核 | 否 | 否 | 是 |
| 预约异常改派/强制取消 | 否 | 否 | 是 |
| 账号管理 | 否 | 否 | 是 |
| 重置密码 | 否 | 否 | 是 |
